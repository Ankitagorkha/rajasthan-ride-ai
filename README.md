# 🐪 Rajasthan Ride AI

**A Premium Rajasthan-Themed Bus Booking Platform**  
Modern, Creative & Fully Functional Capstone Project

## ✨ Key Features

- Beautiful desert orange + fort blue glassmorphism design
- Animated realistic seat map with Framer Motion
- Live bus tracking with moving marker on map
- Full user authentication (Register / Login)
- Personal Dashboard with bookings and e-ticket PDF download
- Responsive design (works on mobile too)

## 🛠 Tech Stack

- **Frontend**: React 18 + Vite + Tailwind CSS + Framer Motion + Leaflet
- **Backend**: Node.js + Express + Prisma + PostgreSQL + Socket.io + JWT
- **Database**: PostgreSQL

## 🚀 Local Setup (Windows)

1. Start PostgreSQL:
   ```powershell
   docker-compose up -d
   ```
